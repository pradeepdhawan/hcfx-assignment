Simple contact management application with dotnet core api, sql server db and angular front end

Manage Contacts

Troubleshooting:

  • Create and Seed Database : Run Update-Database command in Package Manager Console or run dbo.contact.sql file from code
  • Run Server : dotnet run --project .\Contact.Api\Contact.Api.csproj --urls=http://localhost:61976/ after changing directory to root
  • Run Client : ng serve --proxy-config proxy.conf.json and make sure you specify proxy config
